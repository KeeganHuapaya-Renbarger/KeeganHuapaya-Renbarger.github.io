Chess Game

This is a working chess game that you can play by entering moves and pressing enter or clicking the submit button. This program started as a team project for UCI EECS 22L but I wanted to go farther with the project because I was interested in development so I took the terminal game that we created and made it playable on my website.

Some of the main challenges were:
Getting the terminal colors to work (I used ASCII Escape codes for the original terminal game),
Figuring out Emscripten (the converter that allowed my C language program to convert to .html, .js, & .wasm needed to run this terminal game),
& Getting the correct spacing/font/padding to ensure the UI looks correct in the browser.
